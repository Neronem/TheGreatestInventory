## 🧐 Project Introduction
본 프로젝트 **"위대한 인벤토리"**는 간단한 인벤토리 시뮬레이터로,
인벤토리에서 아이템을 장착 후, 상태창에서 자신의 스탯 정보를 확인할 수 있습니다.
<br><br>

## 🎮 조작 방법

메인메뉴
 - Status 버튼 클릭 -> 상태창 진입
 - Inventory 버튼 클릭 -> 인벤토리 진입

상태창
 - 상태 확인 가능
 - 나가기 버튼 클릭 -> 메인 메뉴

인벤토리
 - 보유한 아이템 확인 가능
 - 아이템 클릭 -> 장착 또는 해제
